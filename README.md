#根据赛项要求实现以下基本功能

#访客

阅读博文

评论博文

对博主留言

关注博主、发私信 点赞与打赏

#博主

博文发表、编辑、删除与浏览

博文分类、归档

博文评论管理

留言管理

个人信息维护

对其他博主关注

#管理员

用户管理

博文管理并可以推荐博文自动实现敏感信息过滤

#非功能性需求

系统可以为用户提供较快的响应速度，页面加载速度不应该超过 10 秒;

系统无明显的安全漏洞，包括 SQL 注入和 XSS 跨站脚本注入;

系统应该兼容响应式布局;

系统应该具有较好的可扩展性和可维护性;
