# thinkphp- blog

 
## 使用说明
1.  直接进入网站首页index.php  
       第一次进入时根据提示，新建数据库
2. 后台登录密码默认为admin；

3. 可以在用户管理中；将第三方账号标记为站长；然后后台就必须使用第三方账号登录以增强安全性；

4. 自定义标签 /Application/Common/Tag/My.class.php；
 

## 项目介绍
1. 前台基于boostrap的响应式页面布局适配手机和平板；
2. 带表情的ajax无限级评论系统；
3. PHPMail邮件系统；
4. QQ、微博、豆瓣、人人、开心网等第三方登录；
5. ueditor 百度富文本编辑器；
7. font-awesome；
8. iCheck；
